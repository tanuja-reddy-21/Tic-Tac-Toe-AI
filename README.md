# Tic-Tac-Toe AI with Pygame

An interactive, graphical Tic-Tac-Toe game built in Python using the **Pygame** library. The game features an unbeatable AI opponent powered by the **Minimax Algorithm**.

## Features
- **Graphical User Interface:** Clean and modern look using Pygame graphics.
- **Unbeatable AI:** The AI uses the Minimax decision-making algorithm to ensure it always plays optimally (you can tie or lose, but you can't beat it!).
- **Restart Mechanic:** Easily reset the board at any point during or after a game.

---

## Technical Specifications
- **GUI Canvas Size:** 600x600 pixels
- **Grid Configuration:** Standard $3 \times 3$ matrix
- **Language/Framework:** Python 3, Pygame, Math

---
